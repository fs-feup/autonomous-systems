Perception Package

Package Information

Description

The Perception module is responsible for the processing of the perception module and perception sensors' data and its transformation into useful information. In this case, the LiDAR's point cloud is processed to generate the cones' position on the track.

Folder Structure

  • center_calculation: Functions to calculate the center of the cones
  • cone_differentiation: Functions to differentiate cones of different types
  • cone_validator: Functions to validate cones
  • perception: Perception node class
  • clustering: Functions to create clusters of cones
  • cone_evaluator: Functions to evaluate cone detections' quality
  • ground_removal: Functions to remove the ground from the point cloud
  • utils: Utility functions
  • icp: Functions to perform ICP, used in a validation method

Launch Configurations

  • eufs.launch.py: Launch file for the EUFS simulator
  • rosbag-preprocessed.launch.py: Launch file for the a preprocessed rosbag with a cone scene
  • vehicle.launch.py: Launch file for the 01 vehicle

Important Dependencies

  • Eigen3
  • PCL

How to Run

Install Dependencies

./dependencies_install.sh

Compiling

colcon build --packages-up-to perception

Testing

colcon test --packages-select perception # use event-handler=console_direct+ for imediate output

To check test results:

colcon test-result --all --verbose

Running

Use a launch file:

source ./install/setup.bash # If in a new terminal
ros2 launch perception eufs.launch.py

or run directly:

source ./install/setup.bash # If in a new terminal
ros2 run perception perception
