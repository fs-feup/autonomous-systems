Detailed Description

Adapter class for coordinating communication between different modes and Control module.

Adapter class for coordinating communication between different modes and Planning module.

Adapter class for coordinating communication between different modes and the Perception module.

Class that handles the communication between the SpeedEstimation node and the other nodes in the system according to the selected mode.

This class serves as an intermediary for communication between different modes, such as simulators or lidar itself.
