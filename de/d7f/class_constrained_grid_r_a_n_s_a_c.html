Detailed Description

Implementation of ground removal using a grid-based RANSAC algorithm.

This class inherits from the GroundRemoval abstract class and provides an implementation for ground removal using a grid-based RANSAC algorithm.

Perform ground removal on the input point cloud using grid-based RANSAC.

This method implements the groundRemoval virtual function from the GroundRemoval interface. It segments the input point cloud into grids and applies RANSAC on each grid for ground removal.
