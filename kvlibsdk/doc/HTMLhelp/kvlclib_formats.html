Supported Formats

The following formats are supported by kvlclib. R,W indicates that a converter has read and write capabilities, respectively.

  • KME24 (R/W)
  • KME25 (R/W)
  • KME40 (R/W)
  • KME50 (R/W)
  • Vector BLF FD (R/W)
  • Vector BLF (W)
  • Vector ASCII (R/W)
  • CSV Frame (W)
  • CSV Signal (W)
  • Plain text (R/W)
  • MDF (R/W)
  • MDF Signal (W)
  • MDF v4.1 (R/W)
  • MDF v4.1 Signal (W)
  • Matlab (W)
  • ASCII J1587 (W)
  • FAMOS (W)
  • DIAdem DAT (W)
  • MTS RPC III (W)
  • Converter API (kvlclib)
