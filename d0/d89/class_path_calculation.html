Detailed Description

PathCalculation class for generating local paths.

The PathCalculation class contains methods for calculating the best local path and stores input data and results related to path planning.

Initializes the path using the current pose and the nearest midpoints.

If a precomputed path exists, it snaps to the nearest midpoints and filters out close or already-visited points. Otherwise, it selects the best two starting midpoints based on the vehicle's current pose. Also updates the set of discarded cones along the way.

Parameters
pathOutput vector to store the selected path points.
midPointsVector of available midpoints.
poseCurrent pose of the vehicle.
point_to_midpointMap for fast lookup from CGAL point to midpoint.
visited_midpointsSet of midpoints already used in the path.
discarded_conesSet of cones to discard along the generated path.

Extends the current path by exploring nearby midpoints.

Iteratively adds new midpoints to the path using a DFS-based cost search, respecting a maximum number of points and cost constraints. Updates the set of visited midpoints and discards cones along the extended path.

Parameters
pathOutput path to be extended.
midPointsVector of available midpoints.
point_to_midpointMap for fast lookup from CGAL point to midpoint.
visited_midpointsSet of midpoints already used in the path.
discarded_conesSet of cones to discard along the extended path.
max_pointsMaximum number of new points to add to the path.

Process an array of cones to generate a local path.

This function processes an array of cones representing a track and generates a local path by selecting positions based on certain criteria.

Parameters
cone_arrayPointer to the array of cones representing the track.
Returns
A vector of pointers to PathPoint objects representing the generated path.

The function utilizes Delaunay triangulation (CGAL) and direction-based selection of positions to create a meaningful local path.
